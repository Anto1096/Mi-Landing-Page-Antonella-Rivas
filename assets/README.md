# ASSETS FOLDER

Este directorio contiene todas las imágenes y recursos multimedia del portafolio.

## Estructura recomendada:

```
assets/
├── images/
│   ├── profile-placeholder.jpg      # Foto de perfil (350x350px)
│   ├── project-1.jpg               # Captura proyecto 1 (600x400px)
│   ├── project-2.jpg               # Captura proyecto 2 (600x400px)
│   ├── project-3.jpg               # Captura proyecto 3 (600x400px)
│   ├── testimonial-1.jpg           # Foto testimonial 1 (80x80px)
│   ├── testimonial-2.jpg           # Foto testimonial 2 (80x80px)
│   └── favicon.ico                 # Favicon del sitio
├── documents/
│   └── cv-antonella-rivas.pdf      # CV en PDF
└── icons/
    └── custom-icons.svg            # Iconos personalizados
```

## Especificaciones de imágenes:

### Foto de perfil:
- Formato: JPG o PNG
- Tamaño: 350x350px (cuadrada)
- Peso: máximo 100KB
- Calidad: alta resolución para pantallas retina

### Proyectos:
- Formato: JPG o PNG
- Tamaño: 600x400px (aspect ratio 3:2)
- Peso: máximo 200KB cada una
- Mostrar capturas representativas del proyecto

### Testimonios:
- Formato: JPG o PNG
- Tamaño: 80x80px (cuadradas)
- Peso: máximo 30KB cada una
- Fotos profesionales o avatars

### Favicon:
- Formato: ICO
- Tamaños: 16x16, 32x32, 48x48
- Diseño simple y reconocible

## Recomendaciones:

1. **Optimización**: Usar herramientas como TinyPNG o ImageOptim para comprimir las imágenes
2. **Formatos**: Considerar WebP para mejor compresión en navegadores modernos
3. **Alt text**: Todas las imágenes deben tener texto alternativo descriptivo
4. **Responsive**: Las imágenes se adaptan automáticamente a diferentes tamaños de pantalla
5. **Lazy loading**: Implementado para mejorar el rendimiento de carga

## Notas importantes:

- Las imágenes placeholder actuales deben ser reemplazadas por contenido real
- Verificar que todas las imágenes tengan los derechos de uso correspondientes
- Mantener consistencia en el estilo y calidad de las imágenes
- Considerar crear versiones en diferentes resoluciones para dispositivos móviles
